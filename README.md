# Nachi Clothing 👗

A beautiful, mobile-first ethnic wear store with WhatsApp ordering, Admin Panel, and smooth product browsing.

## Pages
- `index.html` – Home page
- `collection.html` – Product collection & filters
- `product.html` – Single product page
- `admin.html` – Admin Panel (login required)

## Features
- 🛒 Cart with WhatsApp ordering
- 📦 Order Details Form (Name, Phone, City, Pincode)
- 📲 Direct image share to WhatsApp via native share sheet
- 🖼️ Admin: manage products, categories, hero slider, category images
- 🚚 Per-product Free Delivery control
- 🔍 Product search and filtering
